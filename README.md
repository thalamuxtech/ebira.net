Ebira is spoken by ~4 million people, yet has almost no digital footprint — absent from NLLB-200, Common Voice, and FLEURS. Ebira.net is the open platform that changes that: **preserve** the language, **engage** the diaspora, **enable** education and research. A [Haleyouth Foundation](https://haleyouthfoundation.org) programme.

## Features

| Pillar | Status |
|---|---|
| 🏠 Landing — nine pillars, roadmap, waitlist (offline-capable) | ✅ Live |
| 📖 Dictionary — search, dialect filters (Tao · Etuno · Igu), entry pages | 🔍 Preview |
| 🔁 Translator — dictionary-backed EN ↔ Ebira with confidence badges | 🔍 Preview |
| 🎙️ Contribute — consent-first submissions with offline queue | 🔍 Preview |
| 🧩 Heritage · 🎓 Learn · 🏛️ History · 👥 People · 🌍 Diaspora | 🚧 In design |

Light/dark themes · WCAG-minded · tone-mark-safe self-hosted fonts · `prefers-reduced-motion` respected.

## Quick start

```bash
npm install
npm run dev     # http://localhost:3000
npm run build   # production build
```

## Stack

Next.js (App Router) · TypeScript · Tailwind CSS v4 · Framer Motion · Firebase (Auth, Firestore, Storage)

## Governance & licensing

Content is reviewed by native speakers; the Chief Language Advisor (Ebira Development Project, Okene) is the authority on orthography. Contributors consent at submission and are credited.

**Open by default:** data CC BY 4.0 · code Apache 2.0 · dataset cards CC0.
